Matrix Multiplication using CUDA

Optimized matrix multiplication using CUDA C++ with shared memory and tiled computation.

GitHub Documentation

Overview

This project implements dense matrix multiplication on the GPU using CUDA. The implementation compares CPU and GPU execution times while exploring different optimization strategies.

Objectives

  • Implement CPU baseline.
  • Implement CUDA kernel.
  • Use shared memory.
  • Measure execution time.
  • Compute speedup.
  • Validate numerical correctness.
